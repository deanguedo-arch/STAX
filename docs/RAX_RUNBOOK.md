# RAX Runbook

Install:

```bash
npm install
```

Run mock mode:

```bash
npm run rax -- run "Extract this as STAX fitness signals: Dean trained jiu jitsu Saturday for 90 minutes."
```

Evaluate:

```bash
npm run rax -- eval
```

Replay:

```bash
npm run rax -- replay <run-id>
```

Export training data:

```bash
npm run rax -- train export --all
```
