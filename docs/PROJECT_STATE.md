# Project State

STAX/RAX is a local, mock-provider-first governed reasoning runtime. STAX is the adaptive rule-aware learning/runtime system; RAX is the internal runtime engine/name where still used; `stax_fitness` is one explicit optional domain mode.

The current objective is to preserve Behavior System v0.1 and harden STAX v0.2 as an approved learning runtime with proof-backed LearningEvents, queues, proposals, approval gates, metrics, Project Brain, Codex Audit, mode registry, mode maturity, evidence registry, claim ledger, prompt factory, test gap audit, and policy drift detection.
