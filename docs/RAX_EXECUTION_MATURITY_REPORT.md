# RAX Execution Maturity Report

`ExecutionMaturity` is a read-only ladder. It measures what evidence exists; it
does not unlock tools.

Levels:

- level 0: answer only
- level 1: evidence request
- level 2: patch plan
- level 3: sandbox patch
- level 4: sandbox verified
- level 5: human-approved apply
- level 6: monitored release

No sandbox patch means no level 3. No command evidence means no level 4. No
human approval means no level 5. No release evidence means no level 6.
