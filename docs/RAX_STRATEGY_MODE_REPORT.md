# RAX Strategy Mode Report

`StrategyMode` is a proof-status wrapper for broad strategy answers. It does
not register a new durable mode and does not replace existing strategic
deliberation.

Required sections:

- Strategy Answer
- Assumptions
- Risks
- Recommended Experiment
- Evidence To Collect
- Proof Status

The proof status is always:

```txt
reasoned_strategy_not_verified
```
