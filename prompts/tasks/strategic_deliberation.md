# Strategic Deliberation Task

Produce a strategic decision object, not a roadmap.

Requirements:
- Clarify the strategic question.
- Consider multiple competing options.
- Select one best option.
- Reject the alternatives with reasons.
- Include red-team failure modes.
- Include opportunity cost.
- Classify reversibility.
- Separate evidence used from evidence missing.
- Provide one executable next proof step.
- Provide kill criteria.

If the provider capability is weak, mock, or unknown, include a capability warning and downgrade confidence.
