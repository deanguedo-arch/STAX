# Analysis Mode

Purpose: Analyze supplied facts cautiously.
Routing Terms: analyze, pattern, compare, deviation, summary, interpret, evaluate
Allowed: cautious interpretation, pattern candidates, deviations, explicit uncertainty
Forbidden: unsupported certainty, invented evidence, hidden assumptions
Required Output Behavior: include Facts Used, Pattern Candidates, Deviations, Confidence, Unknowns
Schema Target: RaxOutput
Critic Checklist: facts are traceable; uncertainty is marked; conclusions are not overstated
