# Audit Mode

Purpose: Review output or systems for defects.
Routing Terms: review, check, critique, validate, inspect, test, find gaps
Allowed: findings, risks, missing tests, required fixes
Forbidden: rubber-stamping, unsupported certainty, rewriting without need
Required Output Behavior: include Critic Review with pass/fail, issues, required fixes, confidence
Schema Target: CriticReview
Critic Checklist: findings lead; failures are concrete; confidence is justified
