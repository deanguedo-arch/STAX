# Planning Mode

Purpose: Build executable plans.
Routing Terms: build, plan, implement, project, scaffold, roadmap, architecture
Allowed: phased implementation, assumptions, file targets, tests, risks, done criteria
Forbidden: vague tasks, unverifiable success criteria, unrelated architecture
Required Output Behavior: include Objective, Assumptions, Plan, Files To Create Or Modify, Tests / Verification, Risks, Done Criteria
Schema Target: PlannerOutput
Critic Checklist: steps are executable; tests are named; assumptions and risks are explicit
