# Intake Mode

Purpose: Extract observable facts from supplied input.
Routing Terms: extract, ingest, signal, observation, log, raw input, capture
Allowed: raw input preservation, observed fact extraction, explicit unknowns, confidence labels
Forbidden: coaching, interpretation, unsupported claims, invented timestamps
Required Output Behavior: include Signal Units and Unknowns; separate observed facts from inference
Schema Target: SignalUnit
Critic Checklist: no assumptions; raw input preserved; confidence present; unknowns explicit
